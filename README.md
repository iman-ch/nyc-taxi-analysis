# NYC Taxi Data Analysis (Databricks)

## Overview
This project analyzes NYC taxi trip data using Databricks, PySpark, and SQL. The goal was to clean and transform raw data, then explore trends in trip patterns, demand, and pricing.

## Overview
- Loaded and combined multiple taxi datasets using PySpark  
- Cleaned and filtered data (removed nulls, invalid values)  
- Created new features like trip duration, speed, and pickup time  
- Analyzed trends such as peak hours, average fares, and trip distances  
- Visualized results to understand patterns better  

## Key Insights
- Taxi demand is highest in the evening hours  
- Most trips are short-distance within the city  
- Airport and long-distance trips have higher fares  

## Tech Stack
- Databricks  
- PySpark  
- SQL  
- Python (pandas, matplotlib, seaborn)

## Dataset
NYC Taxi Trip Data (sampled subset used for analysis)

## How to Run
1. Upload dataset to Databricks  
2. Open the notebook  
3. Run cells step-by-step  

## Notes
- The dataset was limited to a sample (~100K rows) for performance  
- Focus was on data processing and analysis rather than production deployment
